Question-1: What are the different ways to select an element in the DOM ?

Answer : There are many way to select an element in the DOM. There are the most common
methods :--->

  • By Id : It select the element with the specified Id name and the Id name must be unique.
  • By Class Name : It select a collection of all elements with the same Class name and It use for selcet many elements.
  • By Tag Name : It select all elements with the specified Tag name.
  • By QuerySelector : It selects the first element that matches the specified CSS selector.
  • By QuerySelectorAll : It selects all elements that match the given CSS selector.

    Question-3: What is event delegation in the DOM ?

    Answer : Event delegation is that we attach a single event listener to a parent element instead of multiple child elements.

    Question-4: What is event bubbling in the DOM ?

    Answer : In the DOM Event bubbling is that where an event starts from the target element and reach up to its parent elements..
